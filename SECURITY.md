# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GnosIQ, please report it responsibly.

**Email:** security@gnosiq.ai  
**Subject:** `[SECURITY]` Brief description  
**Response time:** 24 hours

Please do **not** open public GitHub issues for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |

## Scope

- `api.gnosiq.ai` endpoints
- Authentication and authorization
- Data privacy (LGPD/GDPR compliance)
- Cognitive assessment data handling

## Out of Scope

- Issues already publicly disclosed
- Social engineering attacks
- Third-party service vulnerabilities
